# DecodeLabs Internship - Task 2: Exploratory Data Analysis

## Overview
This project covers exploratory data analysis (EDA) on the cleaned sales dataset (1,200 rows, 14 columns) as part of the DecodeLabs Data Analytics Internship.

## Tasks Performed
- Performed outlier detection using the IQR method
- Conducted correlation analysis between numerical features
- Created visualizations (bar charts, histograms, heatmaps, box plots) using matplotlib and seaborn
- Identified key trends and patterns in the sales data

## Tools Used
- Python
- pandas
- matplotlib
- seaborn
- Jupyter Notebook

## Files
- `project task 2.ipynb` - EDA notebook with analysis, code, and visualizations

## Output
Key insights and visual patterns from the dataset, building on the cleaned data from Task 1 and feeding into the SQL-based analysis in Task 3.
